*Assuming you are running these commands at the root of this repo

To run the front-end (client)

1. Unpack client.tar.gz
2. Configure HTTP server to serve client/index.html
3. Run the following in the terminal:

cd src/Frontend
npm install
npm start

Before running any of the following, note:

For each service package
1. Unpack every .tar.gz
2. Change into the extracted directory
3. Run init.sh

To start, run start.sh

To stop, run stop.sh

To run the back-end

cd src/Backend
python3 -m venv venv
source env/bin/activate
pip3 install flask
pip3 install dataset
pip3 install PyJWT

python3 [file].py, where [file] is every file in this folder

Configuration file is at config.py.

To run the media server

cd src/Media
python3 -m venv ./venv
source ./venv/bin/activate
pip3 install -r requirement.txt
./start

Configuration file is at config.py.

Files will be stored in src/Media/files/; database in src/Media/local/
